# Podcast Directory Submission Guide

This document covers how to submit each show in the Tesla Shorts Time podcast
network to the major podcast directories.

## Prerequisites

Before submitting, ensure:
- RSS feed is valid (use [Cast Feed Validator](https://castfeedvalidator.com/))
- At least 3 episodes published
- Podcast artwork meets requirements (3000x3000 px recommended, min 1400x1400)
- Show description is filled in

## Directory Submission Steps

### Apple Podcasts

1. Go to [Podcasts Connect](https://podcastsconnect.apple.com/)
2. Sign in with Apple ID
3. Click "+" to add a new show
4. Paste the RSS feed URL
5. Apple validates the feed — fix any issues
6. Submit for review (typically 24-48 hours)
7. Once approved, the show appears in Apple Podcasts and gets an Apple Podcasts URL

### Spotify

1. Go to [Spotify for Podcasters](https://podcasters.spotify.com/)
2. Sign in or create account
3. Click "Get Started" → "I have a podcast"
4. Paste the RSS feed URL
5. Verify ownership (usually via the email in the RSS feed)
6. Submit — usually live within a few hours

### Google Podcasts / YouTube Music

Google Podcasts was deprecated in April 2024. Podcasts are now on YouTube Music.

1. Go to [YouTube Studio](https://studio.youtube.com/)
2. Navigate to Settings → Channel → Advanced → Podcast RSS feed
3. Paste the RSS feed URL
4. Verify and sync

### Amazon Music / Audible

1. Go to [Amazon Music for Podcasters](https://podcasters.amazon.com/)
2. Sign in with Amazon account
3. Click "Add Your Podcast"
4. Paste the RSS feed URL
5. Verify email ownership
6. Submit — typically live within 24-48 hours

### Podcast Index

1. Go to [Podcast Index](https://podcastindex.org/add)
2. Paste the RSS feed URL
3. Submit — indexed quickly
4. This also enables discoverability in apps that use the Podcast Index API
   (Fountain, Podfriend, CurioCaster, etc.)

### iHeartRadio

1. Go to [iHeartRadio Podcast Submission](https://www.iheart.com/content/submit-your-podcast/)
2. Fill in podcast details + RSS feed URL
3. Submit — review may take 1-2 weeks

### Samsung Podcasts

1. Distributed via Spotify — no separate submission needed
2. If already on Spotify, shows should appear automatically

### Overcast (Manual Add)

Overcast doesn't have a submission portal — listeners can manually add via
RSS URL. Being in Apple Podcasts is sufficient for Overcast discoverability.

### Pocket Casts

1. Go to [Pocket Casts Submit](https://pocketcasts.com/submit)
2. Paste the RSS feed URL
3. Submit — usually indexed within 24-48 hours
